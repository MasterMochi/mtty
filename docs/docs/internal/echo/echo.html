概要

デバイスファイルから読み取ったデータをデバイスファイルに書き込む事でエコーする。データが制御コードの場合は文字変換してエコーする。

機能 - エコー

エコー機能は、デバイス制御から呼び出されて制御コードの文字変換とデバイスファイル書込みを行う。

機能 - エコー - 制御文字変換

データが0x00～0x1Fの場合は下記変換テーブルに従って文字変換を行う。

機能 - エコー - デバイスファイル書込み

デバイスファイルへの書込みにはmvfsライブラリのLibMvfsWrite()を用いて行う。


デバイス書込
関数 LibMvfsWrite()
引数 # 設定値 説明
1 (EchoDo()の引数fd) ファイルディスクリプタ
2 (変数ポインタ) 書込元データ
3 1 無変換時の書込サイズ
(変換後サイズ) 変換時の書込みサイズ
4 (変数ポインタ) 書込結果サイズ
5 (変数ポインタ) エラー要因

グローバル関数 - EchoDo()

関数 EchoDo()
概要 エコー
説明 データをデバイスファイルに書き込む。
データが制御コードの場合は文字変換を行った後に書き込む。
宣言 
#include <Echo.h>

extern void EchoDo( uint32_t    fd,
                    MttyDevId_t id,
                    char        *pData,
                    size_t      size    );
引数 # 引数名 値 説明
1 fd (任意の値) ファイルディスクリプタ
2 id MTTY_DEVID_SERIAL1 /serial1
MTTY_DEVID_SERIAL2 /serial2
3 *pData (任意の値) データ
4 size (任意の値) データサイズ
