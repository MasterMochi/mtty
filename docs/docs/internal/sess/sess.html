概要

ユーザからttyファイルへのセッション管理を行う。セッションはttyファイルオープン時に接続されて、ファイルクローズ時に切断される。ttyファイルオープン時にmvfsによって払い出されたグローバルファイルディスクリプタによってセッションを識別し、グローバルファイルディスクリプタをキーにセッション情報を管理する。

機能 - セッション接続

ttyファイルのオープン時にセッション接続として、セッション情報を生成し管理リストに追加し、VfsOpen応答を送信する。

シーケンスを以下に示す。



初めに、オープンされたファイルパスがmttyで管理するttyファイルか判定する。ttyファイルでない場合はVfsOpen応答を失敗で送信し処理を中断する。


判定ttyファイルパス一覧
# パス 説明
1 /serial1 シリアルポート1
2 /serial1 シリアルポート2

次に、セッション情報のメモリ領域をC標準ライブラリのmalloc()を用いて割り当てる。割り当てが失敗した場合はVfsOpen応答を失敗で送信し処理を中断する。


セッション情報の割り当て
関数 malloc()
戻り値 割り当てたメモリアドレスが返る。
NULLの場合はVfsOpen応答を失敗で送信する。
引数 # 設定値 説明
1 (セッション情報サイズ) セッション情報のサイズ

次に、セッション情報を初期化および設定する。


セッション情報の初期化と設定
項目 設定値 説明
globalFd (SessDoVfsOpen()の引数globalFd) グローバルファイルディスクリプタ
devId MTTY_DEVID_SERIAL1 シリアルポート1。ファイルパスが「/serial1」の場合に設定する。
MTTY_DEVID_SERIAL2 シリアルポート2。ファイルパスが「/serial2」の場合に設定する。
pid (SessDoVfsOpen()の引数pid) 要求元プロセスID

次に、MLibライブラリのMLibListInsertHead()を用いて、セッション情報を管理リストに登録する。


セッション情報の登録
関数 MLibListInsertHead()
戻り値 処理結果を返す。
引数 # 設定値 説明
1 (変数ポインタ) リストハンドル
2 (変数ポインタ) セッション情報

最後に、mvfsライブラリのLibMvfsSendVfsOpenResp()を用いて、VfsOpen応答を行う。


VfsOpen応答
（正常終了時）
関数 LibMvfsSendVfsOpenResp()
引数 # 設定値 説明
1 (SessDoVfsOpen()の引数globalFd) グローバルファイルディスクリプタ
2 MVFS_RESULT_SUCCESS 成功
3 (変数ポインタ) エラー要因

VfsOpen応答
（不正なttyファイルパス、および、セッション情報の割り当てに失敗した場合）
関数 LibMvfsSendVfsOpenResp()
引数 # 設定値 説明
1 (SessDoVfsOpen()の引数globalFd) グローバルファイルディスクリプタ
2 MVFS_RESULT_FAILURE 失敗
3 (変数ポインタ) エラー要因

機能 - セッション切断

ttyファイルのクローズ時にセッション切断として、セッション情報を管理リストから削除し、VfsClose応答を送信する。

シーケンスを以下に示す。



初めに、MLibライブラリのMLibListSearchHead()を用いて、管理リストからグローバルファイルディスクリプタに該当するセッション情報を検索し、削除する。


セッション情報の検索と削除
関数 MLibListSearchHead()
戻り値 セッション情報を返す。
NULLの場合はVfsClose応答を失敗で送信し処理を中断する。
引数 # 設定値 説明
1 (変数ポインタ) リストハンドル
2 (関数ポインタ) セッション情報評価関数
ここで指定した関数はグローバルファイルディスクリプタが一致するか判定する。
3 (SessDoVfsClose()の引数globalFdへのポインタ) セッション情報評価関数への引数。
検索したいグローバルファイルディスクリプタを設定する。
4 MLIB_LIST_REMOVE 一致したセッション情報を管理リストから削除する。

次に、C標準ライブラリのfree()を用いて、グローバルディスクリプタに対応するセッション情報を解放する。


セッション情報の解放
関数 free()
引数 # 設定値 説明
1 (変数ポインタ) セッション情報

最後に、mvfsライブラリのLibMvfsSendVfsCloseResp()を用いて、VfsCloseの応答を送信する。


VfsClose応答
(正常終了した場合)
関数 LibMvfsSendVfsCloseResp()
引数 # 設定値 説明
1 (SessDoVfsOpen()の引数globalFd) グローバルファイルディスクリプタ
2 MVFS_RESULT_SUCCESS 成功
3 (変数ポインタ) エラー要因

VfsClose応答
(セッション情報が無い場合)
関数 LibMvfsSendVfsCloseResp()
引数 # 設定値 説明
1 (SessDoVfsOpen()の引数globalFd) グローバルファイルディスクリプタ
2 MVFS_RESULT_FAILURE 失敗
3 (変数ポインタ) エラー要因

機能 - デバイスID変換

セッション情報からオープンされているttyファイルのグローバルファイルディスクリプタをデバイスIDに変換する。

シーケンスを以下に示す。



初めに、MLibライブラリのMLibListSearchHead()を用いて、グローバルファイルディスクリプタに該当するセッション情報を取得する。


セッション情報の取得
関数 MLibListSearchHead()
戻り値 セッション情報を返す。
NULLの場合は無効デバイスIDを返す。
引数 # 設定値 説明
1 (変数ポインタ) リストハンドル
2 (関数ポインタ) セッション情報評価関数
ここで指定した関数はグローバルファイルディスクリプタが一致するか判定する。
3 (SessConvertFdToId()の引数globalFdへのポインタ) セッション情報評価関数への引数。
検索したいグローバルファイルディスクリプタを設定する。
4 MLIB_LIST_GET 一致したセッション情報を管理リストから取得する。

最後に、取得したセッション情報からデバイスIDを取得してこれを返す。
